# AIUC-1 Control Mapping for SOC 2 Learning Agent

This document maps the AIUC-1 Responsible AI controls to their specific enforcement mechanisms within the SOC 2 Learning Agent project. It is intended for auditors and developers to understand how each control is implemented and verified.

## Scope

This project implements a subset of the 51 AIUC-1 controls that are relevant to a tool-using, Azure-based AI agent. Controls that are purely organizational (e.g., establishing an ethics board) or not applicable to this learning lab are out of scope.

## Enforcement Types

| Type | Description |
|---|---|
| **Architectural** | The control is enforced by code or infrastructure deterministically. The LLM cannot bypass it. |
| **Prompt-Based** | The control is enforced by instructions in the agent's system prompt. Adherence is not guaranteed. |
| **Platform** | The control is enforced by the underlying Azure AI Foundry platform. |
